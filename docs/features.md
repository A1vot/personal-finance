# Функциональность приложения (Features)

## 1. Общее описание

Приложение представляет собой персональный финансовый трекер с поддержкой бюджета, транзакций, категорий и отчётов. Основные функции включают:
- авторизацию и регистрацию пользователей
- управление месячным бюджетом
- добавление доходов и расходов
- фильтрацию операций по периоду
- отчёты по категориям
- удаление транзакций

## 2. Авторизация и регистрация

###  Авторизация

Реализована через **login_view**.

Функции:
- вход по имени пользователя и паролю
- обработка ошибок
- перенаправление в личный кабинет

### Регистрация

Реализована через **register_view**.

Функции:
- создание нового пользователя
- проверка совпадения паролей
- проверка уникальности имени пользователя

## 3. Личный кабинет (Dashboard)

Логика реализована во **dashboard_view**.

### Основные возможности:

### 3.1. Управление бюджетом

- отображение текущего месячного лимита
- изменение лимита через форму
- автоматическое создание бюджета при первом входе
- расчёт остатка бюджета

### 3.2. Управление транзакциями

- добавление доходов и расходов
- выбор категории
- указание суммы, даты и описания
- автоматическая привязка транзакции к пользователю

### 3.3. Категории

Используются глобальные категории из модели **Category**:

- доходы
- расходы

Категории участвуют в отчётах и фильтрации.

### 3.4. Фильтрация по периоду

Фильтр по GET‑параметрам:

```
?start=YYYY-MM-DD&end=YYYY-MM-DD
```

Позволяет:
- просматривать операции за выбранный период
- формировать отчёты по датам

### 3.5. Отчёты

Отчёты формируются по категориям:
- расходы по категориям
- доходы по категориям

Используются агрегирующие функции (`annotate`, `Sum`).

## 4. Удаление транзакций

Реализовано через **delete_transaction_view**.

Функции:
- удаление транзакции по ID
- защита: пользователь может удалить только свои операции
- автоматический возврат в Dashboard

## 5. Админ‑панель

Django Admin предоставляет:
- управление категориями
- управление транзакциями
- управление бюджетами
- управление пользователями

## 6. Тестирование

Каждое приложение содержит тесты:
- `budget/tests`
- `categories/tests`
- `transactions/tests`
- `pages/tests`

Покрываются:
- модели
- формы
- вью

## 7. Итог

Приложение предоставляет полный набор функций для персонального учёта финансов:
- учёт доходов и расходов
- контроль бюджета
- отчёты по категориям
- фильтрация по датам
- удобный интерфейс Dashboard
