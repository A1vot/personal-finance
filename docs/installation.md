# Установка и запуск проекта

## 1. Требования

Для запуска проекта необходимы:

- Python 3.14+
- pip
- виртуальное окружение (`venv`)
- SQLite (идёт в комплекте с Python)
- Git (для клонирования репозитория)

## 2. Клонирование проекта

Склонируйте репозиторий:

```
git clone <URL_репозитория>
cd personal-finance
```

## 3. Создание виртуального окружения

Создайте окружение:

```
python3 -m venv .venv
```

Активируйте:
Linux / macOS

```
source .venv/bin/activate
```

Windows (PowerShell)

```
.venv\Scripts\activate
```

## 4. Установка зависимостей

Установите пакеты из `requirements.txt`:

```
pip install -r requirements.txt
```

Если нужно обновить зависимости — используйте **freeze**.

## 5. Применение миграций

Создайте базу данных и примените миграции:

```
python manage.py migrate
```

## 6. Создание суперпользователя

Для доступа в админ‑панель:

```
python manage.py createsuperuser
```

Введите:
- имя пользователя
- пароль

## 7. Запуск сервера разработки

Запустите локальный сервер:

```
python manage.py runserver
```

После запуска проект будет доступен по адресу:

```
http://127.0.0.1:8000/
```

## 9. Полезные команды

### Проверка стиля кода

```
flake8
```

### Создание резервной копии БД

```
python manage.py dumpdata > backup.json
```

### Восстановление БД

```
python manage.py loaddata backup.json
```

## 10. Завершение работы

Отключение виртуального окружения:

```
deactivate
```
