Справка

Это приложение для управления личными финансами.

Вы можете отслеживать расходы, управлять бюджетом и анализировать свои транзакции.

Для доступа к полному функционалу необходимо авторизоваться.

Как пользоваться приложением

Это ваш личный кабинет для управления финансами. Ниже краткое описание всех разделов.

1. Бюджет
  • В блоке «Бюджет» отображается ваш месячный лимит, потраченная сумма и остаток.
  • Чтобы изменить лимит введите новое значение в поле «Новый лимит» и нажмите «Сохранить».
  • Остаток автоматически обновляется после каждой транзакции.
2. Добавление транзакции
  • В разделе «Добавить транзакцию» выберите категорию, сумму, дату и описание.
  • Нажмите «Добавить», чтобы сохранить транзакцию.
  • Доходы и расходы различаются по категориям, выбирайте нужную.
3. Список транзакций
  • Все операции отображаются в таблице «Транзакции».
  • Длинные описания автоматически сокращаются, но полный текст можно увидеть, наведя курсор.
  • Чтобы удалить запись нажмите кнопку «Удалить» в строке нужной транзакции.
4. Отчёты по периоду
  • В блоке «Отчёты по периоду» выберите даты «С» и «По» и нажмите «Показать».
  • Ниже появятся две таблицы: расходы по категориям и доходы по категориям.
  • Это помогает анализировать, куда уходят деньги за выбранный период.
5. Общая логика работы
  • Все данные сохраняются.
  • Интерфейс минималистичный: только нужные функции без лишних элементов.
  • Вы можете в любой момент изменить бюджет, добавить или удалить транзакцию и сформировать отчёт.
