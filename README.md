# Personal Finance — персональный финансовый трекер

Минималистичное Django‑приложение для учёта личных финансов: бюджет, доходы, расходы, отчёты и фильтры. Проект ориентирован на строгий, чистый интерфейс и прозрачную архитектуру.

## Возможности

Основные функции приложения:

- **Авторизация и регистрация**
- **Месячный бюджет**: установка и изменение лимита
- **Транзакции**: доходы и расходы
- **Категории**: доходы / расходы
- **Фильтрация по периоду**
- **Отчёты по категориям**
- **Удаление транзакций**
- **Админ‑панель Django**

Подробнее см. **features.md**.

## Архитектура

Приложение построено на стандартной структуре Django:

- маршруты — `urls.py`
- логика — `views.py`
- шаблоны — `templates/pages/`
- модели — `budget`, `categories`, `transactions`
- личный кабинет — `dashboard_view`


Полное описание: **architecture.md**.

## Модели данных

Используются три основные сущности:

- **Budget** — месячный лимит пользователя
- **Category** — категории доходов и расходов
- **Transaction** — финансовые операции

Диаграмма связей и описание: **models.md**.

## Установка и запуск

Кратко:

```
git clone <repo>
cd personal-finance
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Полная инструкция: **installation.md**.
